❓ Como Usar

            

                
📌 Adicionar/Editar Dados:

                
  1. Use as abas para navegar entre médicos e pacientes
  2. Clique em "+ Novo" para adicionar um registro
  3. Preencha o formulário e clique em "Salvar"
  4. Para editar, clique no ícone de lápis ✏️
  5. Para excluir, clique no ícone de lixeira 🗑️

                
📌 Sincronizar com Google Sheets:

                
  1. Crie uma planilha no Google Sheets
  2. Adicione as abas "doctors" e "patients"
  3. Publique a planilha (Arquivo > Compartilhar > Publicar na web)
  4. Copie o ID da planilha da URL
  5. Atualize o arquivo js/google-sheets.js com seu Sheet ID
  6. O sistema carregará automaticamente os dados

                
📌 Para usar apenas JSON local:

                
Vá para a aba "Exportar Dados", copie o JSON e cole no arquivo correspondente no GitHub.
