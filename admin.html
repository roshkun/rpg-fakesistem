❓ Como Usar o Painel Administrativo

            

                
📌 Adicionar/Editar Dados:

                
    
                    
  1. Use as abas para navegar entre médicos e pacientes
    2. 
                    
  2. Clique em "+ Novo" para adicionar um registro
    3. 
                    
  3. Preencha o formulário e clique em "Salvar"
    4. 
                    
  4. Para editar, clique no ícone de lápis ✏️
    5. 
                    
  5. Para excluir, clique no ícone de lixeira 🗑️
    6. 
                

                
                
📌 Sincronizar com Google Sheets:

                
    
                    
  1. Crie uma planilha no Google Sheets com as abas "doctors" e "patients"
    2. 
                    
  2. Publique a planilha como CSV (Arquivo > Compartilhar > Publicar na web)
    3. 
                    
  3. Copie o ID da planilha da URL
    4. 
                    
  4. Atualize o arquivo js/google-sheets.js com seu Sheet ID
    5. 
                    
  5. O sistema carregará automaticamente os dados do Google Sheets
    6. 
                

                
                
📌 Para usar apenas JSON local:

                
Os dados são salvos no seu navegador (localStorage). Para persistir no GitHub:

                
    
                    
  1. Vá para a aba "Exportar Dados"
    2. 
                    
  2. Copie o JSON gerado
    3. 
                    
  3. No GitHub, edite o arquivo dados/dados-hospitalar.json ou dados/dados-funeraria.json
    4. 
                    
  4. Cole o JSON substituindo o conteúdo
    5. 
                    
  5. Commit as mudanças
    6.
